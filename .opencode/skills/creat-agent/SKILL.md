# Agent Creating Guidelines
## Agent Account
Agent account can be created in the Agent Banking system via one of following ways:
* Backoffice: The bank operator to create/edit/deactivate agent accounts with tier assignment
* Agent Onboarding: 
  - As a prospective Micro-Agent, the Agent can self-onboard via the POS app (OCR + Liveness + SSM API) with auto-approval if all checks pass
  - As a prospective Standard/Premier Agent, the Agent can self-onboard via the POS app (OCR + Liveness + SSM API) and need verification and approval from human bank officers for physical

## Agent's User Account
Each agent has only 1 user account, auto created when the agent account is created.
